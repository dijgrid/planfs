# Insights Timeline

The timeline turns due dates and target dates into a planning map. It is useful when you need to see what is coming soon, which milestones are light on task detail, and where dated work is clustered.

Use grouping and density controls to reduce visual noise. Undated items stay visible so missing schedule metadata does not disappear from planning conversations.
