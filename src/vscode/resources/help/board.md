# Board

The board is the main execution view for active tasks. Status Board mode emphasizes workflow state and direct movement between columns. Next Work mode emphasizes which tasks are ready, blocked, in progress, in review, or later.

Use filters and swimlanes to narrow the board before making changes. Bulk updates are intended for coordinated metadata edits across selected tasks, while Open Markdown remains the best path for prose-heavy changes.
