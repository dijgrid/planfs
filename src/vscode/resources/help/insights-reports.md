# Reports

Reports summarize the current `.planfs` state for planning review. They help answer whether epics are moving, how work is distributed, and whether missing dependencies are blocking progress.

Export JSON, CSV, or Markdown when you need to share the current planning snapshot or inspect it outside VS Code.
