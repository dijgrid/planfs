# Structured Editor

The structured editor keeps frequent metadata changes fast while preserving the Markdown file as the source of truth. Use compact fields for status, priority, ownership, dates, tags, and relationships.

Common Markdown sections such as Acceptance Criteria and Questions are shown for review. Use Open Markdown when you need to edit the full body, add new sections, or make prose-heavy changes.
