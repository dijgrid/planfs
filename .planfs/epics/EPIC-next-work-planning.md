Make "what should I work on next?" a first-class PlanFS workflow across the core library, CLI, and VS Code board.

The workflow should rank actionable work using generic priority, dependency readiness, due dates, milestone or epic pressure, critical path position, and assignee relevance. Recommendations must be explainable so users can trust why a task appears above another task.
