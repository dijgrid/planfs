# Archive

The archive keeps completed or inactive tasks and epics available without mixing them into active planning views. Archived items remain Markdown files with archive metadata, original paths, and their previous content.

Restore an item when it should return to active planning. Delete only when the archived record is no longer useful, because archive deletion is permanent.
